# Footy Tips

Marketing website for sports-tipping.com.au.

Created using [Directus](https://directus.io) backend and [Nuxt](https://nuxtjs.org) frontend.

## Getting started

### Server (Directus) Setup

```bash
$ cd server
$ npm i
$ npx directus start
```

### Client (Nuxt) Setup

```bash
$ cd client
$ npm i
$ npm run dev
```
